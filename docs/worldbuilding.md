# 世界观与剧情线

## 核心隐喻

「养蛊」——将众多毒虫投入同一密器，相互蚕食，最终存活者吸尽死者之毒，成为蛊中之蛊。

渊就是那个密器。没有人设计了它，没有人选择了谁进去——它只是在那里，而人向它的底部坠落，就像水向低处流。

「蛊虫」是天渊对众人的称谓。他们有资格往下看，有资格这么叫。只是密器从未打开过——所谓「到顶」，不过是密器里另一层位置。蛊中之蛊，还是蛊。

## 机制叙事映射

每个核心机制都有对应的叙事根基，机制设计不得与叙事解释产生矛盾：

| 系统                       | 叙事解释                                                                                                                                           |
| -------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------- |
| 战斗系统                   | 每一局是一次意识凝聚的试炼：SAN 代表清醒与自我，越低越接近渊虫；归零即彻底失去；死亡即涣散，印记不灭——下一轮带着说不清楚来历的直觉，在原点重新凝聚 |
| 元进度系统（技能树解锁）   | 能力不从外部习得——印记凝聚，某一刻被认出，成为可以调用的东西；解锁是找回                                                                           |
| 元进度系统（遗物图鉴）     | 拼凑他人留下的记忆残留，也是在拼凑自己；图鉴满时，最后那块是余烬                                                                                   |
| 全局装备系统               | 意识可以涣散，物件更难；留下的东西比主人更长久地记得自己是什么                                                                                     |
| 成就系统                   | 渊对特定行为的应答——做过了，它记住了；不是奖励，是记录                                                                                             |
| 主线关卡系统（Boss战终局） | 每一圈是渊对参与者身份的逐级检验，向内是接近真相也接近孤独；Boss战是终局相认                                                                       |
| 交易系统                   | 渊里没有匿名的交换——每次都与具体的某个人：掌柜买卖物资，记录者换情报，修补者押注随机结果。对方是谁，决定了这次交易意味着什么；给出去的不能要回来   |

## 世界背景

### 渊是什么

渊就是这里的世界。

不是某处建筑，不是某个更大世界里的隔离区——它是生活在其中的人所能抵达的全部。没有人知道它有没有边界，也没有人走到过边界。它有自己的地理、秩序、经济，以及一段没有起点的历史。

规则不需要写下来。在这套秩序里生活过一段时间的人都知道——不是被告知，是感知到，像感知到重力一样：**这里只有一个上升的位置，其他所有人为那个位置付出代价。**

渊一直在运转。没有人记得它从什么时候开始，也没有人知道是否有人曾经设计了它。它不依赖外力维持——它通过自己的逻辑延续：胜者升至天渊的位置，天渊维持系统运转，系统产生下一个胜者。这套机器不需要意图，只需要结构。

这里的人清醒地走过每一条走廊、每一个战场、每一个事件现场。只有死亡时才会感到「消散」——意识坠入黑暗，下一刻在原点重新醒来，只剩下一些说不清楚从哪里来的、零碎的印记。

### 他们从何而来

他们就生活在这里。

有人生于下层，从来没见过别处。有人从更高的位置跌落——债、罪、或只是运气耗尽——滑入了这套秩序的底部。有人跟着某条路走到了某扇门，走进去，回头门不在了。有人在某个极端的时刻——最绝望、最走投无路的那一秒——意识重新清醒时已经在这里，说不清楚经过了什么。

渊没有招募，没有选拔。它有引力。人向它的最底层坠落，就像人向贫困坠落——通过许多小选择，和许多与选择无关的事情。

没有人在抵达时被告知规则。规则不需要告知。

所谓的「到顶」，没有人需要解释——每个人都知道那是什么。「到顶」通往的不是别处，是这套秩序的最顶端，以及那个位置真正意味着什么。渊没有外面，只有更深和更高。

### 天渊是谁

天渊是**上一轮的胜者**，或者再上一轮，或者再更早。

没有人记得第一位天渊。每一个坐上那个位置的人，都曾经是在渊底拼死求生的人。他们抵达了顶端——然后发现那个位置是另一种囚禁：成为这套系统的执行者，维持它继续运转，等待下一个胜者出现，把位置交接出去。

这套系统不需要外力维持。它通过胜者延续自身。每一个活下来的人都将维持密器继续运转，使下一批人继续向它坠落——因为他们别无选择，或者因为他们已经不再记得曾经有过选择。

> 屠龙者终成恶龙，并非是腐化，而是遗忘。

**天渊的层级**：

| 称号      | 说明                                                                                                 |
| --------- | ---------------------------------------------------------------------------------------------------- |
| 天渊·先遣 | 最近一轮的胜者，尚未完全失去过去的自己，言语间偶尔流露出旧日的困惑。真名**烬**，无人知晓，终局时自述 |
| 天渊·旧主 | 更早的胜者，只记得规则，已遗忘规则存在之前的自己（扩展关卡 Boss）                                    |
| 天渊·本体 | 系统的人格化——无法确认是否还是一个人，还是已经成为了「密器」本身                                     |

## 世界阶层

渊从外向内收窄，像一个密封空间向中心坠落。

人进入渊的方式只有一种：从外围涌入。外围最宽，最混乱，废墟最多，人也最多。越往内，空间越窄，人越少，规则越重。主角从外围出发，每穿越一道隔断，就离真相更近一步，也离人群更远一步。

### 荒渊

渊的外围废墟带。混乱，嘈杂，没有规则——或者说，规则只有一条：活下去。

**新至者**是还没有被磨损的人——记忆完整，SAN 满，对这里的运作方式还没有形成经验。他们有各自的来历，各自落到这里的路，只有一件事是共同的：没有人告诉过他们「到顶」究竟意味着什么。

**渊虫**是这一层的另一种存在。他们曾经也是新至者，但 SAN 在某一刻归零了，意识涣散，只剩下战斗本能。他们不记得自己的名字，不记得自己为什么在这里，只会攻击任何移动的东西。系统不清除他们——他们是荒渊最好的示范，也是最直接的战场。

> 你在荒渊杀死的每一只渊虫，都曾经是一个有名字的人。

### 游渊

荒渊废墟与壁渊隔断之间的游荡地带——活过荒渊、尚未被拦住的人停留在这里。

他们有的在单打独斗，有的在临时结盟，有的在等待时机背刺。这片地带有它自己的地下经济：信息、物资、让路——都可以交换。没有人是真正的盟友，但偶尔也有真正的信任，虽然不持久。

SAN 在这里缓慢下降。见过足够多的东西之后，人会开始用结果合理化手段。多轮存活下来的人，往往比新来的人更危险，也更冷静，也更空洞。

这里偶尔能遇到某个**无名参与者**——存活了太久，开始认不出人，但又没有完全失去自己。他们是外围废墟与壁渊隔断之间最清晰的镜子。

### 壁渊

渊的第一道真正意义上的物理屏障——渊正守在外圈通往内圈的关卡，穿越需要以战胜负。

渊正不参与竞争。他们维持渊的「秩序」。

没有任何文件定义什么是秩序。渊正自己决定。

他们是前参与者，在某一刻被系统「选中」转型——可能是因为某种交换，可能是因为天渊的决定，可能是因为他们自己某一刻放弃了赢的念头并被系统利用了这一点。转型之后，他们不再只是在渊中挣扎求生的人，也不再是普通意义上的人，他们是这套机器的零件，职责是让机器继续运转。

渊正不攻击没有「违规」的人。但他们对违规的定义，只有他们自己知道。

### 驻渊

掌柜和缝合者，以及记录者、修补者。他们有一个在外围流传的称谓：**渊驻**。

不是荣誉，不是头衔——只是一种描述：他们驻留在这里，比任何竞争者都久，用一种不同于竞争的方式活着。渊驻没有自己的组织，也不相互认同这个称谓，只有外人这么叫他们。

他们是渊里最难被定义的存在：不竞争，不执法，不服务于天渊的直接意志。他们只是……在。

系统允许他们存在，因为他们有用——参与者需要补给，需要喘息，需要某种维持战斗状态的条件。彻底消耗掉所有参与者对系统没有好处。

但「有用」只是解释，不是来历。他们为什么在这里，比任何人都久，这个问题没有答案——或者有，但他们都不说。

### 天渊

渊的中心。主角最终要抵达的地方。

天渊·先遣就站在这里，执行他的职务：观察、调整、维持渊的运转，等待下一个胜者的出现，然后把位置交出去。

这里与外围所有地带的区别不是权力，而是**孤独**。到了这里，没有盟友，没有敌人，没有可以打的仗。只有职务，和一些越来越模糊的、说不清楚从哪里来的情绪残留。

## 流派与人生哲学

四种流派不只是战斗方式，是这些人对「如何活」的根本答案。同一种答案在不同层被系统磨压之后，会演化成截然不同的形态——玩家会在五个层次里反复遇见「同一种人」的不同命运。

| 流派     | 战斗逻辑           | 人生哲学                     | 关键词 |
| -------- | ------------------ | ---------------------------- | ------ |
| **均衡** | 攻防兼备，见机行事 | 不把鸡蛋放在同一个篮子里     | 活路   |
| **猛攻** | 主动出击，以快打慢 | 等待就是把主动权交出去       | 先手   |
| **龟缩** | 承受、积蓄、还击   | 让对手先犯错，然后一次收回来 | 忍     |
| **赌狗** | 孤注一掷，博取极值 | 运气也是实力，不赌不知道     | 押     |

## 主要 NPC

### 荒渊

新至者。记忆完整，哲学纯粹，还没有被磨损。每一种流派在这里呈现的是它最干净的原始形态。

---

**通才**（均衡）

前职业已经不重要了，但看他说话的方式，像是做过某种需要综合判断的工作。

他没有急着找人结盟，也没有急着找人开打。他在观察：谁是潜在威胁，哪条路走起来最安全，什么情况下该出手、什么情况下该退。他给自己留了很多退路，也因此总是走在任何决定的后面一步。

「先看清楚再动。动错了比不动还糟。」

他的问题是：留退路的人，有时候走进了一条没有退路的死胡同，才发现自己从来没有练习过怎么孤注一掷。

---

**利刃**（猛攻）

话少，行动多。在荒渊的第一天就已经打了三场。他不在乎对手是谁，他在乎的是先动手还是后动手。

「等人家打过来，主动权就不是你的了。先打，就算打错了，错的也是主动选择，不是被动挨打。」

他不是莽夫——他只是有一套非常坚定的逻辑：主动出击可以定义战场，等待只会把定义权交给对方。这套逻辑在荒渊管用，管用到他有点不知道它的边界在哪里。

---

**石壁**（龟缩）

最沉默的新人。他不躲，但也不主动。你攻击他，他承受；他承受完了，还给你。

「急什么，你打我一下，我记着。等你打累了，我再算账。」

他可能早就习惯了承受——欠债、被压、被推着走。他的龟缩不是胆怯，是一种他用了很久的生存方式。他只是不知道，荒渊的「耗」和他熟悉的那种「耗」不完全是同一件事。

---

**骰子**（赌狗）

他是真正的赌徒，或者说，他来到荒渊之前曾经站在更高的地方——然后他押错了，一路输到这里。他没有把这当成惩罚。

「这把肯定行，感觉来了。」

他对胜算的判断完全依赖直觉，对风险的评估建立在「不赌怎么知道」上。他输了很多次，但每次输了都能很快站起来再押一把。他乐观，不是因为他相信自己会赢，而是因为他相信「这次可能不同」——每次都这么相信。

---

### 游渊

同样的四种人，活过了荒渊之后。哲学还在，但已经被磨损、压缩、扭曲成了另一种形状。

---

**骑墙人**（均衡→灰色）

他活下来了，靠的是从来不明确站队。谁占上风他就靠近谁，谁要倒了他就提前撤。他把这叫做「灵活」，别人叫它「滑头」。

「中间人活得最久，因为两边都不恨他，只是都不信他。」

他知道别人怎么看他。他决定了不在乎。在这里活下去比被人尊重更重要——他是这么告诉自己的。只是偶尔，他会在说完这句话之后沉默一下，像是在等自己真的相信它。

---

**疲惫的刀**（猛攻→惯性）

他还在主动出击，但已经不是选择了，是惯性。他停不下来，不是因为他勇猛，而是因为他不知道停下来之后要做什么。

「我也不知道还要打多少场。就是停不了。」

身上的伤很多，动作快但散，像一把用了太久、有点缺口的刀。他最初的哲学——「主动出击」——在这里已经变成了一种他跑在里面出不来的轮子。

---

**承受者**（龟缩→麻木）

他还活着。就这些。

「忍到最后，不是赢了，是忘了在忍什么。」

他当初的策略是让对手先犯错，然后反击。现在他还在等，但越来越说不清楚在等什么，也越来越不确定等到了之后他还有没有力气反击。他不是放弃了，他只是……磨损了。忍耐的形状还在，内容已经空了。

---

**老赌鬼**（赌狗→豁达）

他输了很多次。每次都押，每次都可能输。他现在反而有一种奇怪的平静。

「输得够多，你就想通了——输得起，才敢赌。赌不起的人，才是真的输了。」

他还是会孤注一掷，但少了在荒渊时的那种「这把肯定行」。他知道可能不行，他还是押。这不是天真，是一种看透之后的选择。有时候他会停下来看着主角，像是在判断「这个人输得起吗」。

---

### 壁渊

四个渊正，各自代表一种流派被系统「制度化」之后的样子。哲学还在，但已经不再是个人选择——它变成了他们履行职务的方式。

---

**渊正·衡**（均衡→中立执法）

他一视同仁。规则对所有人都一样，包括他自己。这不是美德，这是他做这份工的方式。

「偏袒一个人，就是对所有人不公平。我不做那种事。」

他还有一点人性，但已经习惯把它按下去。偶尔他会在执行某个裁决之前停顿一秒，然后继续。那一秒里发生了什么，他自己也不确定。

---

**渊正·压**（猛攻→威慑）

他不等冲突发生，他先制造一种让人不敢冲突的气氛。走进他巡逻的区域，大多数人会自动降低声调。

「让他们怕我，比让他们守规矩省事。效果一样，成本更低。」

他已经不记得自己是怎么变成这样的了。他只知道这套方法有效，有效就继续用。

---

**渊正·壁**（龟缩→不作为）

职责范围内的事，他处理。职责范围外的事，他不看。

「不在我管的事里，我不管。我的事情已经够多了。」

他其实看见了很多。但他选择了一种精确的视野：只看需要看的，其余的不存在。这是一种效率，也是一种自我保护——看见了管不了，不如不看。

---

**渊正·算**（赌狗→规则漏洞）

每条规则都有例外。他比任何人都清楚哪里有例外，以及怎么用这些例外。

「规则不是墙，是迷宫。知道路的人，永远比只会撞墙的人走得远。」

他是四个渊正里最难捉摸的。他不违规——他只是把规则用到了别人意想不到的地方。他什么时候会对你动用「例外」，没有规律可循，这才是他真正危险的地方。

---

### 驻渊

找到了竞争之外的位置。不是赢，不是输，而是找到了一种方式留在这里、同时不再被消耗。各自的流派决定了他们「用什么方式留下来」。

---

**掌柜**（均衡→左右逢源）

没有人知道他叫什么，也没有人问过。「掌柜」够用了。

他在这里的时间比任何在场的人都长。不站队，不表态，谁来都卖，谁来都不多说一句话。他的均衡哲学演化成了一种极致的中立——不是因为他不在乎，而是因为他在乎的东西已经不是这里能给的了。

「你和上一个……算了，没什么。买什么？」

他停顿的那个瞬间，是他还剩下的最后一点什么。

---

**缝合者**（龟缩→停止）

他曾经是参与者。某一次濒死之后，他停下来了——没赢，没死，只是停了，然后开始给路过的人包扎。

他的龟缩哲学走到了终点：最极致的「不主动」，是连比赛本身都退出了。他守着这一小块地方，不进不退。系统允许他存在，因为他有用。他接受这个，因为他已经没有力气再想更多。

「别谢我。我也只是……不知道该干嘛了。」

---

**记录者**（猛攻→主动占据信息）

他不打架了，但他的「主动」从来没有消失。他换了一个战场：信息。

他走遍渊的每一层，把一切都写下来——谁在哪里、谁和谁起了冲突、哪条路昨天走得通今天走不通。他的笔记散落在渊的各处，大多数人不知道那是他写的。

「记下来的东西就是我的。记忆会消失，文字不会。」

他的笔记就是散落在事件场景中那些「遗忘的笔记」。其中有一本的最后几行，写的不像是外人的观察。

---

**修补者**（赌狗→押注他人）

他不再赌自己。他开始赌别人。

「我赌你能活下去。」他看着每一个路过的人说这句话，然后帮他们修一修装备，或者指一条路。有时候赌对了，有时候没有。他接受两种结果。

他活着，靠的是别人的胜率。他把自己的最后一注全押在了「这个世界上还有人能赢」这件事上。这是他唯一还信的东西。

---

### 天渊

走到这里的人，不是最强的，是最后的。他们的流派已经很难辨认——经历了太多层的磨损之后，原来的哲学只剩下轮廓，内容已经被替换了。

---

**天渊·旧主**（流派：已无法辨认）

更早的胜者。他记得所有的规则，记得天渊的职责，记得每一个需要执行的程序。他不记得的是规则存在之前的自己。

他偶尔会做出一些「无法解释的」例外决定——放走某个本该被处理的蛊虫，在某个节点改变一条规则，然后对此没有任何解释。没有人知道这是残存的人性，还是系统自我调整的一部分。

「程序就是程序。程序之外的事，我不处理。」

但他处理了。他有时候会处理。

---

**天渊·先遣**（流派：碎片）

最近的胜者。他遗忘了大部分，留下来的只有职能和一些难以定义的情绪残留：说不清楚是不是真实的焦虑，某些词语触发了就会停顿，以及面对某些特定对手时出现的、短暂的、他自己也注意到但不去深究的熟悉感。

他的台词随元进度更新：

> 早期：「规则很简单。你赢了，你就到那里。别想太多。」  
> 中期：「你又来了。……你是不是开始记得什么了？」  
> 后期：「有时候我也不确定，到底是我在等你，还是你在等我。」  
> 终局前：沉默。他看着你，什么都没说，就开始战斗。

终局相认时，他笑了。带着一点说不清楚是骄傲还是遗憾的、很轻的、玩味的笑。然后开口——只说了一个字。

剧终。

## 伙伴·余烬

### 身份与名称

- **主角**：由玩家命名。进入上层后，对外自称**荒**——来处，也是立场。
- **伙伴**：余烬——应当随天渊的胜利一同消散的东西，没有消散，反而成形了

余烬是天渊·先遣被丢弃的另一块碎片，与主角同源，但被切割的是不同的部分。

- **主角**代表被切掉的「太脆弱」的部分：软弱、犹疑、感同身受、一个未兑现的愿望
- **余烬**代表被切掉的「太锋利」的部分：对系统的质疑与愤怒——那个会问「为什么要这样」的声音

天渊不能质疑自己维持的东西，所以把这个声音切掉了。

三者之间有一条命名的线：烬是本名，余烬是从烬身上切落、却未消散的火，主角是从烬身上切落、却仍在凝聚的柔软。「余烬」这两个字，是字面意义——烬留下的余火。只是这件事，渊里没有人知道。

两块碎片在渊独立成形，相遇时彼此感到一种无法命名的引力与危险。吞噬的冲动以占有感、莫名的敌意、某些时刻想让对方消失的念头浮现——两人都不知道为什么。

**吞噬冲动的本质**：主角想要余烬的驱动力与方向感；余烬想要主角的稳定与根基。两人合在一起才接近完整，但「合在一起」的方式只有一种。

### 在游戏中的存在形式

余烬不参与战斗，不提供功能性服务。余烬出现在**各关卡之间的过渡节点**——每次主角完成一层返回枢纽时，余烬在那里，可以对话。

关系通过对话积累，没有数值化的好感度。**图鉴积满时**，游戏通过叙事手段暗示余烬是最后一块拼图——具体形式待定，可以是主角的记忆闪回、余烬言行中的异常、或某个场景细节的汇聚——引导玩家意识到选择的时刻已到，但不直接说破。

### 成长弧光

```
早期：余烬在渊更久、更强，主动照顾主角
      愤怒与质疑是强大的燃料，余烬知道怎么在这里活
中期：主角成长速度反超，靠收集他人记忆碎片持续生长
      余烬没有这个来源，开始感到某种收窄
晚期：主角反过来照顾余烬，两人都不理解为什么会变成这样
终局：图鉴满，隐藏槽位出现，指向余烬
```

弧光的关键在于时序——终局选择出现时，主角欠过余烬的，余烬欠过主角的，两人之间已经有了真实的重量。

### 终局选择

图鉴达到100%后，出现隐藏槽位，指向余烬。玩家面临二选一：

**选择一：战斗**

战斗开始时出现隐藏被动（属性砍半），玩家可选择是否激活：

```
开启被动
├── 输 → 余烬察觉异常（打赢了一个主动削弱的人）
│        → 不杀主角 → 两人一起面见天渊·先遣
│        [余烬知情，被接住的信任]
└── 赢 → 主角选择不杀余烬（意图从未改变）
         → 并入「放弃」分支，余烬知情
         [余烬知道打过了，主角放了自己]

不开启被动
├── 输 → 余烬不察觉，正常的胜利
│        → 杀死主角 → 余烬成为主角剧情 → 永久封号
└── 赢 → 主角独自面见天渊·先遣
         [干净的胜利，没有余地]
```

**选择二：直接放弃**

不触发战斗，回到游戏。余烬完全不知情，主角独自承担秘密。

**放弃路的触发**：余烬被丢弃的本质——驱动与质疑——让余烬无法在空荡荡的渊停下来。余烬开始主动推着主角往前走，想找到尽头，不知道为什么，就是停不了。主角知道秘密，跟着走。两人一起走进最终关卡，Boss 在那里等着。

```
余烬不知情，被自己的本质推向终点
主角知情，跟随，独自明白这一切意味着什么
```

**五条终路对照：**

| 路径                          | 余烬状态               | 主角带着什么去见Boss         | 能否继续游玩 |
| ----------------------------- | ---------------------- | ---------------------------- | ------------ |
| 开启被动→输→余烬不杀          | 知情，信任被接住       | 赌对了                       | 是           |
| 开启被动→赢→主角不杀→余烬推进 | 知情，知道主角放了自己 | 给了机会对方没抓住，仍放手   | 是           |
| 不开启→赢→独自相见            | 不知情                 | 干净的胜利                   | 是           |
| 直接放弃→余烬推进→Boss等待    | 不知情，被本质驱动     | 独自承担秘密，跟着余烬走进去 | 是           |
| 不开启→输→永久封号            | 不知情，成为主角       | 游戏结束                     | **否**       |

### 相认后的游戏状态

任意相认结局触发后，返回枢纽，图鉴锁定在100%。天渊·先遣在此后的遭遇中行为永久改变——不再假装不认识，台词进入新阶段。渊还在运转，但它与主角之间的关系变了。「继续游玩」是在相认之后继续存在，看这个世界如何回应一个已经被认出的人。

## 理智（SAN）的叙事含义

SAN 不只是一条资源条。

在渊的扭曲压力下，人的意识会逐渐磨损。SAN 越低，意味着一个人越来越依赖本能、越来越放弃思考、越来越像一只真正的渊虫。

- **SAN 满**：清醒地知道自己为什么战斗，还记得自己是谁
- **SAN 中等**：开始用结果合理化手段，记忆变得模糊
- **SAN 极低**：只剩战斗本能，已经不在乎自己是谁了
- **SAN 归零**：不是死亡，而是**成为**——意识彻底融入渊，成为场景的一部分，或者成为下一批人所要面对的渊虫

> 渊虫，或许都是某一轮理智归零的前人。渊正是另一条路——被系统选中，换了一种方式继续在这里。

## 记忆系统

### 遗忘的机制

人每次「死亡」都会被重置。新的一轮开始时，他们站在原点，没有上一轮的技能，没有具体的经历记忆。

但**印记会留下来**。

印记不是语言，不是画面，是某种更深层的东西——一个动作的肌肉记忆，一个选择时的直觉，一段无法解释的情绪。元进度系统在叙事层面代表的正是这种积累：人不知道自己为什么会这样，但就是会了。

### 主角的真实身份

主角不是普通意义上在渊中生活的人。

天渊·先遣成为胜者的代价，是亲手切除了自己身上「使他成为人」的那部分——软弱，犹疑，对他人苦难的感同身受，以及他曾经许下但再也无法兑现的某个愿望。那些东西不能被简单销毁，于是被丢入了密器，沉在渊最深处，随着一轮又一轮的往返慢慢凝聚、成形。

**主角，就是天渊·先遣被丢弃的那部分自己。**

这是渊里唯一一个从未从外部抵达的存在——不是坠入的，不是跌落的，是从这个世界内部生长出来的。这也解释了一切：

- 为什么主角每次「死亡」后仍然回来——碎片不会真正消亡，只会分散再聚
- 为什么印记在积累——碎片在找回自己，越来越完整
- 为什么 SAN 归零意味着彻底失去——凝聚度归零，碎片再次漫散，什么都不剩
- 为什么天渊·先遣在战斗中有时会停顿，像是在辨认什么——他在看自己
- 为什么主角没有旧日可以回去——没有「之前」，只有这里

### 记忆的重建

每一局中散布着他人遗留的碎片：潦草的笔记，壁上的刻痕，事件现场的只言片语。

玩家以为在拼凑「前人的故事」。但拼得越多，画面越清晰，轮廓越来越像一个特定的人——而那个人的行事方式，和天渊·先遣有着说不清楚的重叠。

玩家拼凑的，是自己。玩家认出的，也是自己。

**这件事不会被说破。**

## 遗物系统

遗物是主线战斗的专属奖励，是记忆碎片的物理载体。遗物的战斗加成来自它所承载的意识残留——力量的来源是记忆本身。

### 遗物类型

| 类型       | 来源                                               | 来自谁                             |
| ---------- | -------------------------------------------------- | ---------------------------------- |
| **切割型** | 被主动丢弃的记忆——软弱、犹疑、对他人苦难的感同身受 | 普通人死后释放；随机散落在事件节点 |
| **封存型** | 拼死守护但无力随身携带的记忆                       | 渊正/精英掌握                      |

切割型碎片来自许多不同的人，记忆闪回各自破碎、声调各异。其中有一些比其他的更底层、更私密——那是天渊·先遣亲手丢掉的部分，也是主角的起源。玩家不一定立刻察觉，但对比之下轮廓会浮现。

### 选择机制

每次主线战斗结束，提供**二选一**：

- 类型（切割 / 封存）选择前可见
- 遗物名称与机制加成选择前可见
- 记忆文本（内容）获取前封印
- 选定后不可撤回

### 生命周期

- **当局**：持有遗物获得属性加成，关卡结束后重置
- **全局**：首次获取，遗物记忆文本永久收录进图鉴
- **重复获取**：图鉴已收录的遗物仍可在后续跑程中出现，标注「已收录」；选择后仅享受当局加成，图鉴不重复写入

遗物不影响技能系统（暂定独立）。后续可为特定技能设置「持有某遗物」作为前置条件。

### NPC 碎片池与交互机制

战斗结束后，NPC 仍在场。玩家可选择主动上前交互——遗物不是击败的战利品，是连结的产物。

```
战斗结束
└── 玩家选择是否与 NPC 交互
    ├── 碎片池有余量：随机概率触发对话 → 二选一遗物奖励
    │   └── 未触发：NPC 本次什么都没说，下次再来仍有机会
    └── 碎片池耗尽：复读机响应，无奖励
```

- 随机概率与玩家运气挂钩，符合「运气也是实力的一部分」
- 每个 NPC 拥有固定数量的碎片（全局计数），池子耗尽即永久进入复读机状态
- 复读机状态通过对话本身感知：玩家主动交互，得到空洞重复，与之前有温度的回应直接对比
- **天渊·先遣例外**：无碎片池，永远完整，是渊里唯一还剩灵魂的存在

**具名 NPC 出现规律：**

具名 NPC 与其所属世界阶层绑定。每次跑程中，该层随机抽取 2–3 位 NPC 出现，不全出现。碎片池全局计数——无论哪局遇到、交互是否触发，计数跨局共享。碎片耗尽后，该 NPC 在后续跑程中仍会出现，但只作复读机响应。

### 常驻 NPC 好感度

驻渊四位 NPC（掌柜、缝合者、记录者、修补者）是常驻功能性存在，不参与战斗。他们的记忆碎片通过全局好感度积累解锁：

- 玩家每次与常驻 NPC 交互（购买、休息、对话）积累好感度
- 好感度达到阈值 → 解锁该 NPC 的一片记忆碎片 → 收录图鉴
- 好感度机制与战斗遗物系统平行：战斗靠运气，常驻靠积累

各常驻 NPC 碎片类型由其处境决定：

| NPC    | 碎片类型 | 理由                                       |
| ------ | -------- | ------------------------------------------ |
| 掌柜   | 切割型   | 切掉了「在乎结果」，换来永远中立           |
| 缝合者 | 封存型   | 停下来了，但把「想帮人」死死守住没有丢     |
| 记录者 | 封存型   | 拼命记录一切，抗拒遗忘——封存是他的战场     |
| 修补者 | 切割型   | 把自己的未来和自我保全全切掉，押在别人身上 |

### 图鉴

图鉴是所有遗物记忆碎片的全局档案，跨关卡永久保存：

| 状态   | 可见内容                               |
| ------ | -------------------------------------- |
| 未发现 | 槽位存在，内容全隐藏（仅显示类型轮廓） |
| 已获取 | 名称、机制描述、记忆文本永久解锁       |

图鉴完成度对玩家可见，内容不可见——玩家知道还差几件，不知道差的是什么。

图鉴完成度是元进度的重要组成部分。天渊·先遣的台词阶段由整体元进度层级驱动，图鉴积累是其中的核心权重。

**图鉴 100% 时**：游戏通过叙事方式暗示余烬是最后一块拼图，引导玩家做出终局选择（见伙伴·余烬章节）。相认结局无条件覆盖当局 SAN 状态——元进度积累是跨越多局的长期结果，不被单局的理智状态干扰。

## 记忆碎片内容架构

### 总量规划

| 来源              | 碎片数    | 获取方式                               |
| ----------------- | --------- | -------------------------------------- |
| 荒渊 4 NPC × 3 片 | 12        | 战斗后交互，随机概率                   |
| 游渊 4 NPC × 3 片 | 12        | 战斗后交互，随机概率                   |
| 壁渊 4 NPC × 4 片 | 16        | 战斗后交互，随机概率（封存型多一片）   |
| 驻渊 4 NPC × 5 片 | 20        | 好感度阈值解锁                         |
| 无名者·散落碎片   | ~20       | 事件节点，各线补充说明，不影响主线连贯 |
| 余烬              | 1（特殊） | 图鉴100%叙事触发，相认后文本解锁       |
| **合计**          | **~81**   |                                        |

### 文本规格

每片记忆文本：**60–120 字，无标题，无注释，无出处**。读者不知道是谁说的，也不知道什么时候说的。文字就是那块记忆本身，不是关于那块记忆的描述。

### 切割型 vs 封存型 写法原则

**切割型**（主动丢弃）：**断裂**。句子在中途停下，主语和目的语经常缺失，边缘是锯齿形的——因为那个会在乎的「自己」已经被切掉了。

**封存型**（拼死守护）：**凝固**。句子完整，语气郑重，时态错位——用现在式说着早已结束的事，像一封写了很久、不知道寄给谁的信。

### 驻渊碎片序列

好感度机制决定解锁顺序，五片各自成立，合在一起有弧光：

| 片序  | 好感度阈值 | 叙事位置                             |
| ----- | ---------- | ------------------------------------ |
| 第1片 | 低         | 他们现在是什么样——当下的样子         |
| 第2片 | 中低       | 他们在这里做过的一件具体的事         |
| 第3片 | 中         | 他们曾经还在竞争时的某个瞬间         |
| 第4片 | 中高       | 那个决定性时刻——为什么停下来了       |
| 第5片 | 高         | 最深处的那块：切掉了什么或守住了什么 |

第5片是真正的秘密——他们从没对任何人说过这个，玩家是因为陪了足够久才被告知的。

### 横向叙事线

**线A：有人见过他变成那样之前**

多个 NPC 的碎片从旁观者角度描述同一个人，不出现名字，只有「他」。随碎片积累，轮廓逐渐清晰，最终与天渊·先遣重叠。

「他」的识别细节——这些行为特征同时存在于碎片描述和游戏实际台词 / 动作中，供玩家事后比对：

- **手的习惯**：做决定或停顿时，手指会扣在某个边沿上，停很久
- **用词习惯**：从不说「赢」，说「到那里」；旁观者在某片碎片里注意到了这个细节
- **被问到「之前」时**：停顿，然后把话题转移；这个模式在碎片里被描述，在天渊台词里能验证
- **赢了之后的神情**：「他站在那里，只是在看自己的手。很久。然后他说：就这样。」

**线B：赢了之后**

各层 NPC 的碎片里都有见证过「赢」的瞬间——早期碎片写的是期待，中期写的是错愕，晚期写的只有沉默。

**线C：这些声音听起来像主角**

来自天渊·先遣直接切除部分的碎片，混入匿名切割型里。语调比其他碎片更私密、更底层。玩家最初不会注意，积累后会有一种奇异的熟悉感：这些话好像是自己曾经想过的。

**线D：不要忘记**

封存型碎片中反复出现的主题——记住、别忘、我要带走这个。这是对系统遗忘机制的本能反抗，也是驱动收集行为的叙事动机。

### 余烬碎片·特殊结构

余烬不产生遗物——余烬就是那块碎片，本身在场。

图鉴 100% 时，图鉴最后一格不出现遗物，而是叙事闪回：主角翻看图鉴，最后一格呈现余烬某个之前对话里出现过的细节，这次像是从不同角度看到的。

余烬的文本只在相认结局后解锁，收录进图鉴最后一格：

> （封存型·唯一）
>
> 我不知道我是什么。  
> 但有一件事我确定——  
> 有什么东西是不能丢的。  
> 我不知道那是什么。  
> 但我确定不能丢。
>
> 所以我还在这里。

与线D的「不要忘记」形成共鸣：所有封存型碎片守住的那件事，余烬也不知道，但余烬也在守。

## 技能解锁系统

技能通过全局解锁池管理，未解锁的技能不进入运行内奖励池。解锁来源两条路径：

| 路径         | 机制                          | 特点             |
| ------------ | ----------------------------- | ---------------- |
| **成就系统** | 完成特定里程碑 → 解锁指定技能 | 确定性，目标明确 |
| **抽奖系统** | 消耗货币 → 随机解锁技能       | 随机性，运气参与 |

货币在运行内获取，抽奖系统作为全局货币回收出口。两条路径并行——成就解锁有目标的技能，抽奖解锁意外的技能，「运气也是实力的一部分」在全局层再次兑现。

运行内战斗技能奖励会优先提供「接近解锁」的技能或对应 delta，推动玩家完成全局解锁进度。

## 功能性 NPC 服务

驻渊四位常驻 NPC 各司其职，构成渊的基础经济与叙事支撑。装备为全局持有，不随关卡重置。

| NPC        | 核心服务 | 说明                                                                     |
| ---------- | -------- | ------------------------------------------------------------------------ |
| **掌柜**   | 买卖两端 | 售卖装备/道具；回收玩家持有物资换货币                                    |
| **缝合者** | 修复     | 恢复 HP / SAN；清除负面状态；低代价或免费                                |
| **记录者** | 情报     | 花费货币预览封印的图鉴槽位内容；揭示地图节点；揭示敌人属性与碎片池剩余量 |
| **修补者** | 赌注     | 抽奖（消耗货币，随机全局解锁技能）；装备强化（消耗货币，随机结果）       |

**货币流向：**

```
运行内获取货币
    ├── 掌柜（买装备 / 卖物资，双向流动）
    ├── 记录者（消耗：购买情报）
    └── 修补者（消耗：抽奖 / 装备强化）

缝合者不参与货币经济
```

记录者的「预览图鉴内容」是打破「选前封印」规则的唯一合法途径——需要付出代价，且只能看，不能替代选择。

## 事件节点

事件节点是蛛丝马迹与前人残留的主要落脚点，承担叙事功能，不与遗物系统竞争资源产出。

- **主要产出**：叙事内容（线索、场景碎片、前人遗留的只言片语）
- **次要产出**：玩家做出选择后触发的后果（货币 / 消耗品 / SAN 变化）
- **不产出遗物**：遗物专属战斗 NPC 交互，职责不重叠
- **偶尔涉及常驻 NPC**：部分事件与特定常驻 NPC 关联，触发好感度累积

事件内永远有选择，选择结果不透明——「决定了就不能后悔」在叙事层兑现。

## 剧情线

### 关卡结构

主线以渊的世界地图为骨架，从外圈向中心顺序解锁，每个地带是一个独立的可重复 Roguelike 跑程：

```
荒渊（外围废墟带）  → 突破，解锁 →
游渊（游荡地带）    → 突破，解锁 →
壁渊（渊正壁垒带）  → 强行穿越，解锁 →
驻渊（内驻地带）    → 突破，解锁 →
天渊（中心）        ← 终局，Boss 战
```

每个地带可重复挑战。突破即解锁相邻内圈，已解锁的地带可自由选择重入。跑程内的遭遇（战斗/事件/常驻NPC节点）保持 Roguelike 随机性。

**货币来源：**

- 主要：击败敌人掉落
- 次要：事件选择结果偶尔产出
- 转换：向掌柜出售/回收物资（存量转化，非净增）

### 主线结构

每一局是一次凝聚与涣散，每一次通关是一次「到顶」，但真正的剧情在多局之间、在元进度的积累中展开。

叙事沿两个维度推进：**关卡推进**（玩家到达哪里）与**元进度积累**（玩家看见多少）。前者决定空间位置，后者决定叙事深度。

**关卡推进（地带对应叙事位置）：**

```
荒渊：活下去
  └── 初入渊。目标明确：击败天渊·先遣，赢得唯一生存名额
        └── 第一次见到天渊·先遣：「你的前辈们也走到这里来过。」

游渊：你不是第一个
  └── 遇见曾经活过、仍然有意识的人
        └── 散碎线索开始出现：他们来自哪里，走向了哪里

壁渊：谁在维持这一切
  └── 首次遭遇渊正，理解渊「秩序」从何而来
        └── 封存型遗物开始出现——有人曾拼死守住什么，没有丢掉

驻渊：这套系统为什么能运转
  └── 见到常驻的掌柜、缝合者等存在，理解渊的支撑结构
        └── 好感度积累解锁记忆碎片：他们为什么在这里，比死更持久

天渊：这里的终点通往哪里
  └── Boss 战。天渊·先遣就在那里，位置相同，姿态相同
        ├── 图鉴未满 → 通关 → 循环延续结局
        └── 图鉴已满 → 终局选择触发（见伙伴·余烬章节）
```

**元进度积累（跨局叙事深度）：**

```
元进度·低
  └── 渊是透明的博弈系统，规则明确，目标明确

元进度·中
  └── 碎片线索开始聚焦——主角的来历，天渊的过去

元进度·高
  └── 轮廓越来越清晰：有些事说不清从哪里来，但它在那里

图鉴 100%
  └── 余烬是最后一块拼图
        └── 玩家面临终局选择，天渊·先遣进入终局前状态
```

具体线索内容见「蛛丝马迹」章节；终局选择细节见「伙伴·余烬」章节；各结局含义见「结局分支」章节。

### 蛛丝马迹

真相不会被说出，但它留有痕迹。每一条线索单独看都可以被合理解释，只有当玩家把它们放在一起时，轮廓才会浮现。

**天渊·先遣的台词变化**（随元进度层级提升，同一场战斗开始前）：

> 早期：「规则很简单。你赢了，你就到那里。别想太多。」  
> 中期：「你又来了。……你是不是开始记得什么了？」  
> 后期：「有时候我也不确定，到底是我在等你，还是你在等我。」  
> 终局前：沉默。他看着你，什么都没说，就开始战斗。

**散布在场景中的细节**（部分需元进度解锁，部分每局随机出现）：

| 细节                   | 描述                                                                                 | 暗示方向                          |
| ---------------------- | ------------------------------------------------------------------------------------ | --------------------------------- |
| 前人的刻痕             | 墙角有人用指甲刻下的字，辨认出来是一个愿望。你有一种奇异的熟悉感，像是说过同样的话。 | 主角与天渊共享某段过去            |
| 渊正的眼睛             | 击败渊正后，他倒下前看了你一眼。不是敌意，不是求饶。是认出了什么的眼神。             | 渊正有记忆残留                    |
| 镜子事件               | 某面镜子里映出的轮廓与你的动作有一帧的延迟，但姿态更像另一个人。                     | 主角是「某人」的投影              |
| 遗忘的笔记（深层文本） | 笔记的最后几行写着：「我开始不记得我丢掉了什么。只知道有什么东西是我不该丢的。」     | 天渊·先遣的内心独白以笔记形式残留 |
| 技能的回响             | 某些技能在第一次使用时，描述里会多出一行只闪过一帧的文字，消失前来不及读完。         | 主角的技能是天渊旧有记忆的残影    |
| 渊虫的犹豫             | 极低概率，普通敌人在攻击前会停顿一瞬，像是认出了什么，然后继续战斗。                 | 场景中的存在与主角有某种连结      |
| 商店店主的话           | 「你和上一个……算了，没什么。买什么？」 无论如何追问，都只有「算了，没什么。」        | 他人隐约察觉但不说破              |

**终局前的确认**（高元进度触发，不可重复）：

> 主角在某面损毁的镜子前停下。  
> 镜中的轮廓，和天渊·先遣战斗时的姿态，一模一样。  
> 没有文字说明。场景继续。

### 结局分支

| 结局          | 触发方式                                   | 叙事含义                                                                                        |
| ------------- | ------------------------------------------ | ----------------------------------------------------------------------------------------------- |
| **循环延续**  | 普通通关，图鉴未满                         | 主角赢得名额，成为下一轮执行者，再次开始遗忘。密器里又有了新的碎片在漂浮。                      |
| **主动遗忘**  | 低 SAN 通关                                | 碎片在最后关头再度涣散——赢了，但已经不记得为什么要赢了。天渊·先遣的表情没有变化，只是转过身去。 |
| **理智归零**  | SAN 归零失败                               | 凝聚了太久的碎片彻底化开，融入场景，成为下一批人要面对的「敌人」。循环得到了新的材料。          |
| **永久封号**  | 与伙伴战斗，不开启被动，失败               | 主角死亡，碎片彻底消散。伙伴不知情，成为新的主角，独自走向天渊·先遣。游戏不可继续。             |
| **相认·独行** | 与伙伴战斗，不开启被动，胜利               | 干净的胜利。主角独自面见天渊·先遣。                                                             |
| **相认·共行** | 与伙伴战斗，开启被动，失败；伙伴察觉，不杀 | 赌对了。两人一起走进最后那扇门。                                                                |
| **相认·等待** | 开启被动赢后不杀伙伴，或直接放弃           | Boss主动出现。两种境地：伙伴知情（战斗后）或茫然（直接放弃）。                                  |

### 终局：相认

天渊·先遣就站在那里，与每一次的位置相同，与每一次的姿态相同。只是这一次，他在门槛被踏入的瞬间停止了移动。

没有独白。没有揭示。他看着走进来的人——或两个人——然后笑了。

不是嘲弄，不是释然，不是悲悯。是那种只有见到某件失而复得的东西时才会有的表情——带着一点说不清楚是骄傲还是遗憾的、很轻的、玩味的笑。

**这个笑在不同的相认版本里意味着不同的东西，但文本从不解释。**

然后，在所有版本里，有一件事相同：他开口了，说了一个字。

> 「烬。」

文本不解释这个字的意思。走进来的人，各自带着自己知道的东西去理解它。

---

**相认·独行**（不开启被动→赢）

主角一个人走进来。他带着一场干净的胜利，和一个不在场的伙伴。Boss的笑认出的是一个完整的、决绝的胜者。

**相认·共行**（开启被动→输→伙伴不杀）

两个人一起走进来。其中一个试图输掉，另一个察觉了，选择不杀。Boss的笑认出的是两块碎片——以及它们之间发生了什么。

**相认·等待·知情**（开启被动→赢→主角不杀→Boss来）

两个人一起站在那里，一个知道打过了什么，另一个知道自己被放过了。Boss主动走来。他的笑里多了什么，说不清楚。

**相认·等待·不知情**（直接放弃→Boss来）

两个人站在那里，一个独自承担着秘密，另一个茫然不知。Boss主动走来。他的笑对两个人意味着完全不同的东西——只有主角同时看见了两层。

**剧终。**

---

剩下的由玩家决定：那个笑是什么意思，走进去之后发生了什么，这一切有没有改变任何事。文本不回答，也不需要回答。玩家带着什么来，就带着什么走。

## 叙事语调

- **克制**：不作宏大陈述，世界的黑暗通过细节渗透
- **模糊**：外部世界永远在背景里，不解释，让玩家自己脑补
- **有余温**：天渊·先遣不是纯粹的反派，渊正不是纯粹的工具，渊里的人不是纯粹的受害者
- **自洽**：所有文本——事件描述、商店说明、技能名称——都在同一个世界里发声，不出戏
